# 文件组织规范

## 路由文件

**路由组件放在 `src/routes` 目录**，使用 TanStack Router 的 FileRoute 模式：

```typescript
// src/routes/chatbot.tsx
import { createFileRoute } from '@tanstack/react-router'
import { ChatbotPage } from '@/pages/chatbot/chatbot-page'

export const Route = createFileRoute('/chatbot')({
  component: () => <ChatbotPage />,
})
```

带子路由的路由文件：

```typescript
// src/routes/books.tsx
import { createFileRoute, Outlet } from '@tanstack/react-router'
import { AppLayout } from '@/components/layouts/app-layout'

export const Route = createFileRoute('/books')({
  component: () => (
    <AppLayout>
      <Outlet />
    </AppLayout>
  ),
})
```

> **注意**：chatbot 页面在页面组件内部自行包裹 `AppLayout`，而非在路由文件中包裹。

## 页面组件

**页面组件放在 `src/pages` 目录**，按路由路径组织：

- 路由 `/dashboard` → `src/pages/dashboard/`（路由文件直接组合子组件）
- 路由 `/books` → `src/pages/books/books-page.tsx`
- 路由 `/categories` → `src/pages/categories/categories-page.tsx`
- 路由 `/chatbot` → `src/pages/chatbot/chatbot-page.tsx`
- 路由 `/customers` → `src/pages/customers/customers-page.tsx`
- 路由 `/products` → `src/pages/products/products-page.tsx`
- 路由 `/orders` → `src/pages/orders/orders-page.tsx`

**命名规范**：页面组件以 `-page.tsx` 结尾

## 页面组件拆分

当页面组件过大需要拆分时，在当前页面目录下创建 `components` 子目录：

```
src/pages/chatbot/
├── chatbot-page.tsx          # 主页面组件
└── components/
    ├── menu-bar.tsx          # 菜单栏
    ├── prompt-input.tsx      # 输入框
    ├── section-card.tsx      # 对话 section 卡片
    ├── section-index-dialog.tsx
    ├── section-index-popover.tsx
    └── generative/           # 生成式 UI 组件
        ├── confirmation-card.tsx    # 操作确认卡片
        ├── missing-fields-form.tsx  # 缺失字段表单
        ├── operation-result-card.tsx # 操作结果卡片
        ├── order-detail-card.tsx   # 订单详情卡片
        ├── order-list-card.tsx     # 订单列表卡片
        └── record-list-card.tsx    # 记账记录列表卡片
```

## 通用组件

**通用组件放在 `src/components` 目录**：

- `src/components/ui/` - shadcn/ui 基础组件（~55）
- `src/components/layouts/` - 布局组件
- `src/components/ai-elements/` - AI 聊天相关组件（~48）

通用组件不应与特定页面耦合，应可被多个页面复用。

## 路径别名

**始终使用 `@/` 前缀引用 src 目录下的模块**：

```typescript
// ✅ 正确
import { Button } from '@/components/ui/button'
import { chat } from '@/api/commands'
import type { ChatMessage } from '@/types/chatbot'

// ❌ 错误
import { Button } from '../../../components/ui/button'
import { chat } from '../../api/commands'
```
