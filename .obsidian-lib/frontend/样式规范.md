# 样式规范

## Tailwind CSS v4

项目使用 Tailwind CSS v4，采用 CSS-first 配置（无 `tailwind.config.js` 文件）。

主题配置位于 `src/styles/tailwind.css`：

```css
@import 'tailwindcss';
@import 'tw-animate-css';
@import 'shadcn/tailwind.css';

@custom-variant dark (&:is(.dark *));

@theme inline {
  /* 从 CSS 变量映射到 Tailwind tokens */
  --color-background: var(--background);
  --color-foreground: var(--foreground);
  /* ... */
}

:root {
  /* 亮色主题 oklch 颜色值 */
  --background: oklch(1 0 0);
  --foreground: oklch(0.141 0.005 285.823);
  /* ... */
}

.dark {
  /* 暗色主题 oklch 颜色值 */
  --background: oklch(0.141 0.005 285.823);
  --foreground: oklch(0.985 0 0);
  /* ... */
}
```

## 使用 Tailwind 类名

```typescript
// ✅ 使用 Tailwind 类名
<div className="flex h-screen bg-background p-4">
```

## cn 工具函数

使用 `cn()` 合并条件类名（基于 clsx + tailwind-merge）：

```typescript
import { cn } from '@/lib/utils'

<div className={cn(
  'base-classes',
  isActive && 'active-classes',
  className
)}>
```

## 主题支持

所有组件必须支持亮色和暗色主题，使用语义化的颜色类：

```typescript
// ✅ 使用语义化颜色（自动适配主题）
<div className="bg-background text-foreground border-border">
<div className="bg-card text-card-foreground">
<div className="bg-muted text-muted-foreground">

// ❌ 避免硬编码颜色（不会随主题变化）
<div className="bg-white text-black">
<div className="bg-gray-100 text-gray-900">
```

### 语义化颜色 token

| Token                                            | 用途             |
| ------------------------------------------------ | ---------------- |
| `bg-background` / `text-foreground`              | 页面背景和主文字 |
| `bg-card` / `text-card-foreground`               | 卡片背景和文字   |
| `bg-popover` / `text-popover-foreground`         | 弹出层           |
| `bg-muted` / `text-muted-foreground`             | 次要/辅助信息    |
| `bg-primary` / `text-primary-foreground`         | 主要操作按钮     |
| `bg-secondary` / `text-secondary-foreground`     | 次要操作         |
| `bg-destructive` / `text-destructive-foreground` | 危险/删除操作    |
| `border-border`                                  | 边框             |
| `ring-ring`                                      | 焦点环           |
| `bg-accent` / `text-accent-foreground`           | 悬停高亮         |

## 主题切换

通过 `next-themes` 管理主题（`src/components/theme-provider.tsx`）：

```typescript
import { ThemeProvider } from '@/components/theme-provider'

// 在根路由中包裹
<ThemeProvider defaultTheme="system" storageKey="theme">
  <Outlet />
</ThemeProvider>
```

## 响应式设计

使用 Tailwind 的响应式断点：

```typescript
// 移动端优先
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3">

// 仅移动端显示
<nav className="md:hidden">

// 移动端隐藏
<aside className="hidden md:block">
```
