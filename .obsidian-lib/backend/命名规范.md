# 命名规范

## 结构体/枚举

使用 **PascalCase**：

```rust
// 服务
pub struct AccountingService { }
pub struct CustomerService { }
pub struct OrderService { }
pub struct CategoryService { }

// 枚举（统一在 enums/ 模块下）
pub enum AccountingType { }
pub enum AccountingChannel { }
pub enum AccountingRecordState { }
pub enum CustomerCategory { }
pub enum OrderType { }
pub enum OrderStatus { }
pub enum OrderSubType { }
pub enum MessageRole { }
pub enum MessageState { }

// DTO
pub struct AddAccountingRecordDto { }
pub struct UpdateAccountingRecordDto { }
pub struct CreateCustomerDto { }
pub struct CreateOrderDto { }
pub struct SettleOrderDto { }
pub struct CreateCategoryDto { }

// 实体 Model
pub struct Model {
    // ...
}
```

## 函数/方法

使用 **snake_case**：

```rust
// 服务层 CRUD 操作标准命名
pub async fn create_record(&self, input: Dto) { }   // 创建
pub async fn get_record(&self, id: i64) { }         // 获取单个
pub async fn get_all_records(&self) { }              // 获取全部
pub async fn update_record(&self, input: Dto) { }   // 更新
pub async fn delete_record(&self, id: i64) { }      // 删除
pub async fn search_records(&self, keyword: String) { } // 搜索

// 批量操作
pub async fn batch_post_records(&self, ids: Vec<i64>) { }

// 关联查询
pub async fn get_records_by_order_id(&self, order_id: i64) { }
pub async fn get_settle_preview(&self, order_id: i64) { }

// 初始化函数
pub fn new(db: DatabaseConnection) -> Self { }

// 实体方法
pub async fn generate_id<C: ConnectionTrait>(db: &C) -> Result<i64, ...> { }
```

使用 **SCREAMING_SNAKE_CASE**：

const DEFAULT_BOOK_ID: i64 = 2021010100001;

## 方法命名约定

### 服务层方法

| 操作类型 | 方法名模式            | 示例                                                |
| -------- | --------------------- | --------------------------------------------------- |
| 创建     | `create_xxx`          | `create_record`, `create_customer`                  |
| 查询全部 | `get_all_xxxs`        | `get_all_customers`, `get_all_products`             |
| 查询单个 | `get_xxx_by_id`       | `get_record_by_id`, `get_customer_by_id`            |
| 搜索     | `search_xxxs`         | `search_customers`, `search_products`               |
| 更新     | `update_xxx`          | `update_record`, `update_customer`                  |
| 删除     | `delete_xxx`          | `delete_record`, `delete_customer`                  |
| 条件查询 | `get_xxxs_by_yyy`     | `get_orders_by_status`, `get_records_by_order_id`   |
| 批量操作 | `batch_xxx`           | `batch_post_records`                                |
| 预览     | `get_xxx_preview`     | `get_settle_preview`                                |
| 检查     | `is_`, `has_`, `can_` | `is_running`, `has_records`                         |

### 命令层函数

与 Tauri 前端调用的函数名保持一致，使用 snake_case：

```rust
#[tauri::command]
pub async fn create_accounting_record(...) { }

#[tauri::command]
pub async fn update_accounting_record(...) { }

#[tauri::command]
pub async fn get_all_customers(...) { }
```

使用单个大写字母或描述性名称：

```rust
// 常见约定

E  // 错误
K  // 键
V  // 值
```
