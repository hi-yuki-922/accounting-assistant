# 命令层 (Commands)

## 命令定义

统一使用 `#[tauri::command]` 宏定义 Tauri 命令：

```rust
use tauri::State;

use crate::services::CustomerService;
use crate::entity::customer;
use super::dto::*;

/// 创建客户
#[tauri::command]
pub async fn create_customer(
    service: State<'_, CustomerService>,
    input: CreateCustomerDto,
) -> Result<customer::Model, String> {
    service.create_customer(input)
        .await
        .map_err(|e| e.to_string())
}
```

> **规范**：所有命令必须使用 `#[tauri::command]` 全限定注解，不使用 `#[command]` 短写法。

## 状态注入

### 服务注入

通过 `State<'_, ServiceType>` 注入服务实例：

```rust
#[tauri::command]
pub async fn get_all_products(
    service: State<'_, ProductService>,
) -> Result<Vec<product::Model>, String> {
    service.get_all_products()
        .await
        .map_err(|e| e.to_string())
}
```

### AppHandle 注入

需要访问应用资源时注入 `AppHandle`（如文件操作）：

```rust
use tauri::AppHandle;

#[tauri::command]
pub async fn create_attachment(
    app: AppHandle,
    service: State<'_, AttachmentService>,
    master_id: i64,
    file_name: String,
    file_content: Vec<u8>,
) -> Result<(i64, String), String> {
    service.create_attachment(&app, master_id, file_name, file_content)
        .await
        .map_err(|e| e.to_string())
}
```

## 参数处理

### 基本类型

```rust
#[tauri::command]
pub async fn delete_customer(
    service: State<'_, CustomerService>,
    id: i64,
) -> Result<bool, String> {
    service.delete_customer(id)
        .await
        .map(|_| true)
        .map_err(|e| e.to_string())
}
```

### DTO 类型

```rust
#[tauri::command]
pub async fn create_order(
    service: State<'_, OrderService>,
    input: CreateOrderDto,
) -> Result<OrderDetail, String> {
    service.create_order(input)
        .await
        .map_err(|e| e.to_string())
}
```

### 可选参数

```rust
#[tauri::command]
pub async fn search_customers(
    service: State<'_, CustomerService>,
    keyword: Option<String>,
) -> Result<Vec<customer::Model>, String> {
    let keyword = keyword.unwrap_or_default();
    service.search_customers(keyword)
        .await
        .map_err(|e| e.to_string())
}
```

```rust
pub async fn get_all_customers(
    service: State<'_, CustomerService>,
```

```rust

    service: State<'_, CustomerService>,
```

## 命令注册

在 `commands/mod.rs` 中集中注册所有命令：

```rust
mod accounting;
mod accounting_book;
mod attachment;
mod chat;
mod customer;
mod order;
mod product;

pub fn with_install_tauri_commands(
    builder: tauri::Builder<tauri::Wry>
) -> tauri::Builder<tauri::Wry> {
    builder.invoke_handler(tauri::generate_handler![
        // 记账相关
        accounting::create_accounting_record,
        accounting::modify_accounting_record,
        accounting::post_accounting_record,
        // ...

        // 账本相关
        accounting_book::create_book,
        accounting_book::get_books,
        // ...

        // 附件相关
        attachment::create_attachment,
        attachment::delete_attachment,
        // ...

        // 聊天相关
        chat::create_chat_session,
        chat::get_all_chat_sessions,
        // ...

        // 客户相关
        customer::create_customer,
        customer::get_all_customers,
        // ...

        // 商品相关
        product::create_product,
        product::get_all_products,
        // ...

        // 订单相关
        order::create_order,
        order::settle_order,
        // ...
    ])
}
```

## 命名约定

命令函数名使用 snake_case，与前端调用保持一致：

```rust
// Rust 命令定义
#[tauri::command]
pub async fn create_customer(...) { }

// 前端调用
invoke('create_customer', { input: data })
```

## 错误处理

命令层统一将服务层的 `Box<dyn std::error::Error>` 转换为 `String`：

```rust
service.method(params)
    .await
    .map_err(|e| e.to_string())
```

删除操作额外映射为 `bool`：

```rust
service.delete_xxx(id)
    .await
    .map(|_| true)
    .map_err(|e| e.to_string())
```
