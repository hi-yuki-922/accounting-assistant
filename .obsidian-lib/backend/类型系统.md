# 类型系统

## 金额类型

### 必须使用 Decimal

```rust
use rust_decimal::Decimal;

// ✅ 正确：实体定义
#[sea_orm(column_type = "Decimal(Some((19, 4)))")]
pub amount: Decimal,

// ✅ 正确：从 f64 转换
let amount_decimal = Decimal::from_f64_retain(self.amount)
    .ok_or_else(|| "金额格式无效".to_string())?;

// ✅ 正确：计算
let total = amount1 + amount2;
let tax = amount * Decimal::from_str("0.1")?;

// ❌ 错误：永远不要使用浮点数
pub amount: f64,  // 禁止！
let total = 10.0 + 20.0;  // 禁止用于金额！
```

### Decimal 操作

```rust
use rust_decimal::Decimal;
use std::str::FromStr;

// 从字符串创建
let amount = Decimal::from_str("123.45")?;

// 从整数创建
let amount = Decimal::from(100);

// 从 f64 创建（保留精度）
let amount = Decimal::from_f64_retain(123.456)
    .ok_or("转换失败")?;

// 算术运算
let sum = a + b;
let diff = a - b;
let product = a * b;
let quotient = a / b;  // 注意除法精度

// 四舍五入
let rounded = amount.round_dp(2);

// 比较
if a > b { }

// 格式化
let display = amount.to_string();
```

## 日期时间

### 数据库存储

使用 `NaiveDateTime`（不带时区）：

```rust
use chrono::NaiveDateTime;

pub struct Model {
    pub record_time: NaiveDateTime,
    pub create_at: NaiveDateTime,
}
```

### 应用使用

使用 `chrono::Local` 本地时间：

```rust
use chrono::Local;

// 获取当前时间
let now = Local::now().naive_local();

// 解析日期时间
let dt = NaiveDateTime::parse_from_str("2024-01-15 10:30:00", "%Y-%m-%d %H:%M:%S")?;

// 格式化
let formatted = dt.format("%Y-%m-%d %H:%M:%S").to_string();
```

### 常用操作

```rust
use chrono::{NaiveDateTime, Duration};

// 时间计算
let tomorrow = dt + Duration::days(1);
let hour_ago = dt - Duration::hours(1);

// 比较
if dt1 < dt2 { }

// 从时间戳创建
let dt = NaiveDateTime::from_timestamp_opt(timestamp, 0)?;
```

## ID 类型

### 实体主键

使用 `i64`（基于日期序列号的 ID）：

```rust
#[sea_orm(primary_key, auto_increment = false)]
pub id: i64,
```

### ID 生成

标准格式为 `YYYYMMDDNNNNN`（8位日期 + 5位序列号），详见 [[实体层#ID 生成]]。

### 关联 ID

```rust
pub struct Model {
    pub id: i64,              // 主键
    pub book_id: Option<i64>, // 外键
    pub write_off_id: Option<i64>,
}
```

## Option 处理

### 可空字段

```rust
pub struct Model {
    pub remark: Option<String>,
    pub write_off_id: Option<i64>,
    pub book_id: Option<i64>,
}
```

### Option 操作

```rust
// 解包 Option
let value = option.ok_or("值不存在")?;

// 提供默认值
let value = option.unwrap_or(default_value);
let value = option.unwrap_or_else(|| compute_default());

// 条件处理
if let Some(v) = option {
    // 有值
} else {
    // 无值
}

// 映射
let new_option = option.map(|v| transform(v));

// 链式调用
let result = option
    .and_then(|v| some_operation(v))
    .ok_or("处理失败")?;
```

### ActiveModel 中的 Option

```rust
// 插入时设置值
active_model.remark = ActiveValue::Set(Some("备注".to_string()));
active_model.remark = ActiveValue::Set(None);  // 明确设为 NULL

// 不设置（使用默认值）
active_model.remark = ActiveValue::NotSet;
```

## 类型转换

### DTO 到实体

```rust
impl CreateDto {
    pub fn to_internal_types(&self) -> Result<(Decimal, NaiveDateTime, Type), Box<dyn std::error::Error>> {
        // f64 -> Decimal
        let amount = Decimal::from_f64_retain(self.amount)
            .ok_or("金额格式无效")?;

        // String -> NaiveDateTime
        let time = NaiveDateTime::parse_from_str(&self.record_time, "%Y-%m-%d %H:%M:%S")
            .map_err(|_| "日期格式无效")?;

        // String -> Enum
        let type_enum = self.accounting_type.parse::<AccountingType>()
            .map_err(|_| "类型无效")?;

        Ok((amount, time, type_enum))
    }
}
```

### 实体到 DTO

```rust
impl From<Model> for AttachmentInfo {

        Self {

        }
    }
}
```

### Vec

let records: Vec<Model> = Entity::find().all(&db).await?;

// 转换
