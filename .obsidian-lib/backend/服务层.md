# 服务层 (Services)

## 服务模块结构

每个服务模块统一使用以下结构：

```
services/
├── [模块名称]/
│   ├── mod.rs      # 导出服务实现和 DTO
│   ├── service.rs  # 具体服务实现
│   └── dto.rs      # DTO 定义
```

### mod.rs 模板

```rust
// services/customer/mod.rs
pub mod dto;
pub mod service;

pub use service::CustomerService;
pub use dto::*;
```

### service.rs 模板

```rust
// services/customer/service.rs
use sea_orm::DatabaseConnection;

use crate::entity::customer;
use super::dto::*;

/// 客户服务
#[derive(Debug)]
pub struct CustomerService {
    db: DatabaseConnection,
}

impl CustomerService {
    pub fn new(db: DatabaseConnection) -> Self {
        Self { db }
    }

    /// 创建客户
    pub async fn create_customer(
        &self,
        input: CreateCustomerDto,
    ) -> Result<customer::Model, Box<dyn std::error::Error>> {
        let id = customer::Model::generate_id(&self.db).await?;
        // ...
    }

    /// 获取所有客户
    pub async fn get_all_customers(
        &self,
    ) -> Result<Vec<customer::Model>, Box<dyn std::error::Error>> {
        customer::Entity::find()
            .order_by_desc(customer::Column::CreateAt)
            .all(&self.db)
            .await
            .map_err(|e| e.into())
    }
}
```

### dto.rs 模板

```rust
// services/customer/dto.rs
use serde::{Deserialize, Serialize};

/// 创建客户 DTO
#[derive(Deserialize, Serialize)]
pub struct CreateCustomerDto {
    pub name: String,
    pub category: String,
    pub phone: Option<String>,
    pub address: Option<String>,
    pub remark: Option<String>,
}

/// 更新客户 DTO
#[derive(Deserialize, Serialize)]
pub struct UpdateCustomerDto {
    pub id: i64,
    pub name: Option<String>,
    pub category: Option<String>,
    pub phone: Option<Option<String>>,
    pub address: Option<Option<String>>,
    pub remark: Option<Option<String>>,
}
```

## 服务结构

每个服务持有数据库连接：

```rust
#[derive(Debug)]
pub struct XxxService {
    db: DatabaseConnection,
}

impl XxxService {
    pub fn new(db: DatabaseConnection) -> Self {
        Self { db }
    }
}
```

> **注意**：`ChatService` 额外派生了 `Clone`。

## 方法命名

### CRUD 操作标准命名

| 操作 | 方法名模式 | 示例 |
|-----|-----------|------|
| 创建 | `create_xxx` | `create_customer`, `create_order` |
| 查询全部 | `get_all_xxxs` | `get_all_customers`, `get_all_products` |
| 查询单个 | `get_xxx_by_id` | `get_customer_by_id` |
| 搜索 | `search_xxxs` | `search_customers`, `search_products` |
| 更新 | `update_xxx` | `update_customer`, `update_product` |
| 删除 | `delete_xxx` | `delete_customer` |
| 条件查询 | `get_xxxs_by_yyy` | `get_orders_by_status`, `get_orders_by_customer_id` |
| 复合查询 | `query_xxxs` | `query_orders` |

### 方法签名

```rust
// 创建 — 返回创建的实体
pub async fn create_customer(&self, input: CreateCustomerDto) -> Result<Model, Box<dyn std::error::Error>>

// 查询全部 — 返回列表
pub async fn get_all_customers(&self) -> Result<Vec<Model>, Box<dyn std::error::Error>>

// 查询单个 — 返回单个实体
pub async fn get_customer_by_id(&self, id: i64) -> Result<Model, Box<dyn std::error::Error>>

// 搜索 — 返回匹配列表
pub async fn search_customers(&self, keyword: String) -> Result<Vec<Model>, Box<dyn std::error::Error>>

// 更新 — 返回更新后的实体
pub async fn update_customer(&self, input: UpdateCustomerDto) -> Result<Model, Box<dyn std::error::Error>>

// 删除 — 返回空
pub async fn delete_customer(&self, id: i64) -> Result<(), Box<dyn std::error::Error>>
```

## 业务规则验证

```rust
pub async fn modify_record(&self, input: ModifyDto) -> Result<Model, Box<dyn std::error::Error>> {
    // 获取现有记录
    let record = Entity::find_by_id(input.id)
        .one(&self.db)
        .await?
        .ok_or("记录不存在")?;

    // 验证业务规则：只有待入账状态可修改
    if record.state != AccountingRecordState::PendingPosting {
        return Err("只有待入账状态的记录可以修改".into());
    }

    // 执行更新...
}
```
