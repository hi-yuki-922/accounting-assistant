# 技术栈

## 前端

### 核心框架

- **React 19** with TypeScript
- **Tailwind CSS v4**（CSS-first 配置，无 tailwind.config.js）
- **Vite**（构建工具和开发服务器，端口 1420）
- **shadcn/ui**（组件库，基于 Radix UI Nova 主题）

### 路由与状态

- **@tanstack/react-router**（文件路由模式）

### AI 集成

- **Vercel AI SDK v6**（AI Agent 框架，ToolLoopAgent）
- **zhipu-ai-provider**（智谱 AI 模型接入，glm-4.7 / glm-4-flash）

### 数据可视化

- **recharts**（图表库，用于仪表盘收入趋势图等）

### UI 增强组件

- **@xyflow/react**（AI Canvas 流程可视化）
- **@dnd-kit/core + @dnd-kit/sortable**（拖拽排序，用于账本管理）
- **cmdk**（命令面板）
- **sonner**（Toast 通知）
- **vaul**（抽屉组件）
- **react-day-picker**（日历选择）
- **react-resizable-panels**（可调整大小的面板布局）
- **input-otp**（验证码输入）
- **embla-carousel-react**（轮播）
- **@base-ui/react**（Combobox 等基础原语）

### AI 聊天界面

- **streamdown + @streamdown/cjk + @streamdown/code + @streamdown/math + @streamdown/mermaid**（Markdown 流式渲染）
- **shiki**（代码块语法高亮）
- **@rive-app/react-webgl2**（Rive 动画，用于 AI 助手形象）
- **ansi-to-react**（终端输出渲染）
- **react-jsx-parser**（JSX 预览）
- **use-stick-to-bottom**（对话自动滚动）
- **tokenlens**（Token 用量追踪）

### 工具库

- **zod**（Schema 验证，用于 AI 工具参数定义）
- **neverthrow**（Result 类型错误处理）
- **radash**（工具函数）
- **date-fns**（日期格式化）
- **nanoid**（ID 生成）
- **@fontsource-variable/geist**（Geist Variable 字体）

### 主题

- **next-themes**（亮/暗主题切换）

## 后端

- **Tauri 2.0**（桌面应用框架）
- **Sea-ORM 2.0-rc** with SQLite（ORM + 数据库）
- **Tokio**（异步运行时）
- **Rust Decimal**（金融精度计算，所有金额字段必须使用）
- **Chrono**（日期时间处理）
- **Serde**（JSON 序列化/反序列化）
- **thiserror**（错误处理派生宏）
- **strum**（枚举工具宏）

### Tauri 插件

- **tauri-plugin-fs**（文件系统访问，用于 JSONL 对话存储）
- **tauri-plugin-opener**（文件/URL 打开）

## 开发工具

- **Ultracite**（零配置代码质量工具，基于 Oxlint + Oxfmt）
- **pnpm**（包管理器）
- **terser**（生产构建压缩）
- **tw-animate-css**（Tailwind 动画工具）
- **code-inspector-plugin**（开发模式代码检查）
- **@egoist/tailwindcss-icons + @iconify/json**（Tailwind 图标集成）
