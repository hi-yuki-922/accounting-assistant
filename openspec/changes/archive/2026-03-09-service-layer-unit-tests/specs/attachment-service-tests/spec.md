## ADDED Requirements

### Requirement: 附件服务支持查询附件信息

测试 SHALL 验证 AttachmentService 能够查询附件的元数据信息。

#### Scenario: 查询附件列表
- **WHEN** 调用列表查询方法，不指定过滤条件
- **THEN** 系统返回所有附件的列表
- **AND** 每个附件包含基本的元数据信息

#### Scenario: 分页查询附件
- **WHEN** 调用查询方法，指定页码和每页数量
- **THEN** 系统返回对应页的附件列表
- **AND** 列表数量不超过每页数量限制

#### Scenario: 按 master_id 查询附件
- **WHEN** 调用查询方法，指定 master_id
- **THEN** 系统返回指定 master_id 的附件列表
- **AND** 所有附件的 master_id 与指定值一致

#### Scenario: 按文件名查询附件
- **WHEN** 调用查询方法，指定文件名模糊匹配
- **THEN** 系统返回文件名包含指定字符串的附件列表

#### Scenario: 按文件后缀查询附件
- **WHEN** 调用查询方法，指定文件后缀
- **THEN** 系统返回指定文件后缀的附件列表

#### Scenario: 按时间范围查询附件
- **WHEN** 调用查询方法，指定开始时间和结束时间
- **THEN** 系统返回创建时间在该范围内的附件列表
- **AND** 不包含时间范围之外的附件

#### Scenario: 查询结果按创建时间倒序排列
- **WHEN** 调用查询方法
- **THEN** 返回的附件列表按创建时间倒序排列
- **AND** 最新创建的附件排在最前面

### Requirement: 附件服务支持删除附件

测试 SHALL 验证 AttachmentService 能够删除附件记录。

#### Scenario: 根据 ID 删除存在的附件
- **WHEN** 调用删除方法，传入存在的附件 ID
- **THEN** 系统尝试删除附件记录
- **AND** 数据库中可能不再包含该附件（取决于物理文件是否删除成功）

#### Scenario: 根据路径删除存在的附件
- **WHEN** 调用删除方法，传入存在的附件路径
- **THEN** 系统成功删除附件记录
- **AND** 数据库中不再包含该附件

#### Scenario: 删除不存在的附件
- **WHEN** 调用删除方法，传入不存在的附件 ID 或路径
- **THEN** 系统返回错误或静默失败
- **AND** 不抛出未处理异常

### Requirement: 附件服务支持查询附件元数据

测试 SHALL 验证 AttachmentService 能够查询不存在的附件。

#### Scenario: 下载不存在的附件
- **WHEN** 调用下载方法，传入不存在的附件 ID
- **THEN** 系统返回错误
- **AND** 错误信息为"附件不存在"
- **AND** 不抛出未处理异常

#### Scenario: 查询空附件列表
- **WHEN** 数据库中不存在任何附件
- **AND** 调用查询方法
- **THEN** 系统返回空列表
- **AND** 列表长度为 0

**注意**：以下功能由于服务实现限制，测试中未实现：
- 文件上传功能（服务需要 Tauri AppHandle）
- 文件下载成功测试（需要真实文件系统操作）
- 文件类型验证（服务未实现）
- 文件大小限制（服务未实现）
- 附件与会计记录关联（AttachmentService 未提供相关方法）
